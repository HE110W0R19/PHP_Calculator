# Документация для Супер мега крутого калькулятора 3000 killer 228 avtoritet

## Структура проекта

Проект разделен на несколько файлов для улучшения организации кода:

- **Calc.php**: Главный файл, отображающий страницу и подключающий внешние файлы.

- **calculator.php**: Отвечает за обработку данных формы, вычисление результата и отображение на странице.

- **functions.php**: Содержит функцию `calculate`, выполняющую арифметические операции.

- **style.css**: Внешний файл стилей для улучшения визуального оформления страницы.

- **script.js**: JavaScript-файл, обеспечивающий отправку данных формы на сервер без перезагрузки страницы.

## Использование

1. **Открытие страницы**:

    - Откройте файл `Calc.php` в веб-браузере.

2. **Ввод данных и вычисление**:

    - На открывшейся странице вы увидите простую форму калькулятора.
    - Введите числа в соответствующие поля.
    - Выберите оператор из выпадающего списка.
    - Нажмите кнопку "Посчитать".

3. **Получение результата**:

    - Результат операции будет отображен на той же странице без перезагрузки.

4. **Использование калькулятора многократно**:

    - Калькулятор может использоваться многократно без необходимости перезагрузки страницы.

## Дополнительные сведения

- Функции калькулятора описаны в файле `functions.php`.

- Стили страницы определены в файле `style.css`.

- Обработка данных формы и отправка на сервер без перезагрузки выполняется с использованием JavaScript в файле `script.js`.

## Примечание

Для наилучшего опыта использования рекомендуется использовать современные веб-браузеры, такие как Google Chrome, Mozilla Firefox, Safari или Microsoft Edge.
